Especificación de caso de uso: Login.


1. Descripción Básica
En este caso de uso se describe todo el procedimiento que un usuario (administrador, trabajador de soporte, empleado) de la facultad de sistemas debe realizar para identificarse dentro del sistema.

2. Flujo Básico
1. Este caso de uso es iniciado por un usuario.
2. El usuario ingresa en la seccion de login en la página inicial.
3. El sistema muestra en pantalla 2 campos para ingresar texto, en donde uno de ellos está destinado para el ingreso del correo de usuario y el siguiente es para la contraseña del mismo.
4. El usuario ingresa el email con el cual está registrado y su correspondiente contraseña.
5. El sistema captura la información ingresada por el usuario.
6. El sistema valida los datos que han sido digitados.
7. El sistema compara la información ingresada con la que está almacenada para encontrar la cuenta del usuario.
8. El sistema muestra al usuario las diferentes opciones que puede realizar una vez identificado.

3. Flujo Alterno
3.1 Nombre de usuario no encontrado:
3.1.1 El sistema no encuentra el nombre de registro almacenado que fue ingresado por el usuario.
3.1.2 El sistema pide al usuario que vuelva a ingresar los datos solicitados anteriormente.
3.1.3 El sistema muestra los campos para el ingreso del nombre de registro y contraseña para realizar la validación con los datos almacenados.

3.2 Contraseña Inválida.
3.2.1 El sistema compara la información ingresada por el usuario con la información que se encuentra almacenada.
3.2.2 El sistema le muestra al usuario un mensaje en donde le dice que la contraseña ingresada para ese correo de registro no es válida.
3.2.3 El sistema muestra los campos para el ingreso del correo de registro y contraseña para realizar la validación con los datos almacenados.



4. Precondiciones
4.1 Registro:
El usuario debe estar previamente registrado en el sistema con sus datos personales y role dentro del sistema.

5. Postcondiciones
5.1 Visualizacion de panel para la gestion de laboratorios:
Permite visualizar los laboratorios con su estado.

5.14 Visualizacion de Opcion de Apertura:
Permite visualizar la opcion para la apertura de nuevos laboratorios
